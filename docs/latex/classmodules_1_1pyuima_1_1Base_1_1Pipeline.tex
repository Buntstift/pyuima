\subsection{\-Detailed \-Description}
\-Basic text processing pipeline. 

\begin{DoxyRefDesc}{\-Todo}

change 'fileids' to a flexible parameter \end{DoxyRefDesc}
\-This class implements a rudimentary \-U\-I\-M\-A pipeline. \-The set-\/up comprises three steps\-: \-First, a text corpus is initialized. \-In addition, optional annotations can be specified through a pandas \-Data\-Frame. \-Second, functions that make use of the whole corpus are set. \-Third, functions that extract certain features are given. \-Finally, the pipeline can be run. \-The results can be obtained by several functions according to the preferred format.
